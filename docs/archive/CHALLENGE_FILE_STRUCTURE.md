## ğŸ”„ HOW FILES WORK TOGETHER

### **Daily Workflow:**

```
Morning:
1. User opens app
2. Sees current challenge in sidebar
3. Checks BACKLOG for today's tasks

During Day:
4. Works on tasks from BACKLOG
5. Marks completed in BACKLOG
6. Takes notes in ACTIVITY LOG

Evening:
7. Does Daily Check-in (via UI or chat)
8. System writes to ACTIVITY LOG
9. Updates streak in challenge-config.json
10. Checks if punishment should trigger
11. Moves completed BACKLOG items to "Done"
```

### **Weekly Workflow:**

```
End of Week:
1. System generates WEEKLY SUMMARY in ACTIVITY LOG
2. User reviews PLAN progress
3. Updates BACKLOG priorities for next week
4. Archives completed BACKLOG items
```

### **Monthly Workflow:**

```
End of Month:
1. Review PLAN milestones
2. Update challenge-config.json if timeline changes
3. Generate progress report from ACTIVITY LOG
4. Adjust PLAN if needed
```
